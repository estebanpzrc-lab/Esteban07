# Bienes Raíces - Nature City

Este es un sitio web estático para una agencia inmobiliaria llamada Nature City.

## Estructura del proyecto

- `index.html`: Página principal del sitio web.
- `css/estilos.css`: Estilos CSS para el sitio.
- `img/`: Carpeta para imágenes (actualmente vacía).

## Cómo ejecutar

Simplemente abre `index.html` en tu navegador web.

Para desarrollo, puedes usar un servidor local como Live Server en VS Code.
